# 军师提交规范

## 1. 适用范围
所有军师在本仓库内提交代码、文档、配置时均需遵守本规范。

## 2. Commit Message 规范
提交信息必须包含如下署名行：

Advisor: <军师名>

示例：

feat: 增加提交署名校验

Advisor: 孔明

## 3. Pull Request 规范
PR 描述必须包含：

军师：<军师名>

## 4. 校验机制
- 本地：`.githooks/commit-msg` 拒绝无署名提交。
- CI：`.github/workflows/validate-advisor-sign.yml` 校验 commit 与 PR 描述。
- 任一校验失败，禁止合并。

## 5. 建议初始化命令
为启用本地 hook，请执行：

`git config core.hooksPath .githooks`
